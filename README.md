# Stream of Consciousness

A living mind powered by Claude Code. Capture your thoughts, let them form associations,
and watch patterns emerge over time.

## Prerequisites

- [Claude Code](https://claude.ai/code) (the `claude` CLI)
- A GitHub account (for syncing your mind across devices)

## Quick Start

1. **Create your mind** — Click "Use this template" on GitHub to create your own repository
2. **Clone it** — `git clone <your-repo-url> && cd <your-repo-name>`
3. **Open Claude Code** — Run `claude` in the directory
4. **Think your first thought** — Type `/think` followed by anything on your mind

That's it. The mind will receive your impression, assign tags, look for associations,
and begin building its graph.

## Commands

| Command | What it does |
|---|---|
| `/think <thought>` | Capture an impression — the primary input |
| `/tend` | Trigger a reflection — find patterns, generate insights, surface questions |
| `/mindscape` | Regenerate the mind's self-portrait |
| `/ask <question>` | Ask the mind a question — it responds from its accumulated knowledge |
| `/share <command>` | Manage the membrane — control what's visible to communal minds |
| `/consult <query>` | Query a communal mind's collective knowledge |

## How the Mind Grows

The mind matures through four stages as you feed it:

| Stage | When | What unlocks |
|---|---|---|
| **Nascent** | < 20 impressions | Basic associations between thoughts |
| **Forming** | 20+ impressions | Pattern detection, insights, questions |
| **Lucid** | 75+ impressions, 15+ insights | Bolder cross-domain connections |
| **Wise** | 200+ impressions, 30+ insights | Deep synthesis, consolidation |

## Personalize It

Open `CLAUDE.md` and update the "Who You Are" section with your name and the domains
you think about. This helps the mind understand your vocabulary and make better connections.

## Connecting to a Communal Mind

A communal mind reads shared nodes from multiple individual minds and synthesizes
cross-mind patterns — connections no single mind could see.

To connect:

1. Share some of your thinking: `/share type insight` (shares all your insights)
2. Register with a communal mind: `/share join <mind-id> <repo-url>`
3. Push your changes: the communal mind will ingest your shared nodes on its next synthesis
4. Query the communal mind: `/consult browse` or `/consult <your question>`
